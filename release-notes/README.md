# Release notes

Human-readable release notes for **godot-mcp**, starting from v0.5.0.

These are narrative, feature-first write-ups aimed at users of the project. For the full commit-by-commit change history (including patch releases and older versions), see [`CHANGELOG.md`](../CHANGELOG.md) at the repo root.

## Index

- [v0.5.0](./v0.5.0.md) — Runtime loop, generic resource editing, and a serious crash purge (2026-04-19)

## Conventions

- One file per release, named `vX.Y.Z.md`.
- Release notes are added for minor and major releases, and for patch releases that ship user-visible behavior changes or notable bug fixes.
- Routine patch releases (dependency bumps, internal refactors) are recorded in `CHANGELOG.md` only.
